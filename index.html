Bootcamp 2020

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of arrays, loops, and randomness. I used the bezier function to create a sort of boomerang type shape. To make it more interactive I decided to use the mouse functions as a way to get Co-ordinates for the shapes. I ended up with a spiral which the user could then zoom in/ bring closer together and then zoom out of just by moving his mouse acorss the screen. I then used the randomness function to find random colours for the lines so that it would also look more lively.
